== Description ==

In an effort to provide all requried tools at one place Currently Available funtionalities includes Adding Google analytics code, Google Tag Manager Code, Search console verification code, custom code add in head, custom code add in body and Read more / read less functionalty. 



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/one-stop-seo` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->One Stop SEO screen to configure the plugin
